# imc-scarlet
IMC Scarlet is not just a theme; it's a performance-first ecosystem, Designed by imc.ge, it eliminates the "bloat" of traditional WordPress themes by offloading core assets to a global CDN (via GitHub & jsDelivr).

It provides a "clean canvas" for Elementor, ensuring your sites load faster, rank higher on SEO, and remain easy to maintain across multiple client projects.
